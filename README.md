# Playvox Workflow

## Tools
* Backend: Flask + mongodb
* Frontend: pending

## Run stack

```bash
docker-compose -f docker-compose.yml up
```
Now you can see the services running:
* Backend: http://127.0.0.1:5000/
* Frontend: pending
